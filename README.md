# Personal Portfolio Website

A modern, responsive personal portfolio website built with HTML5 and CSS3. This project showcases your profile, skills, projects, and contact information with a clean, professional design and mobile-friendly layout.

## Features

- **Responsive Design:** Looks great on desktop, tablet, and mobile devices
- **Sticky Header:** Navigation bar with hamburger sidebar menu on mobile
- **Hero Section:** Eye-catching introduction with your name, title, and a call-to-action button
- **About Section:** Brief bio and a grid of your key skills
- **Projects Section:** Grid layout for featured projects (add your own!)
- **Contact Section:** Invitation to connect
- **Footer:** Social media links (GitHub, LinkedIn, Twitter, etc.) and copyright
- **No JavaScript Required:** All interactivity and responsiveness handled with pure CSS

## Folder Structure

```
Frontend/
├── asset/           # (Place your images here, e.g., profile photo)
├── index.html       # Main HTML file
├── style.css        # Main CSS file
└── README.md        # Project documentation
```

## Getting Started

1. **Clone or Download the Repository**
2. **Open `Frontend/index.html` in your browser**
   - Or use a local server (recommended for best results):
     - With Python 3: `python -m http.server 8000` (from the `Frontend` folder)
     - Open `http://localhost:8000` in your browser
3. **Customize Content**
   - Update your name, bio, skills, and project details in `index.html`
   - Add your own images to the `asset/` folder and reference them as needed
   - Update social links in the footer

## Customization
- **Colors & Fonts:** Easily change in `style.css` using CSS variables
- **Projects:** Add or remove project cards in the Projects section
- **Profile Image:** Place your image in `asset/` and update the hero section if desired
- **Social Links:** Update the URLs in the footer for your own profiles
